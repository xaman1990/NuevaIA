# Pilot Offer — Control Package (v1)

## Oferta
Implementación de Agent OS para **1 workflow crítico** con foco en control operativo y auditoría.

## Alcance
- Registro/versionado de 1–3 agentes.
- Allowlist de tools y políticas básicas.
- Runs con trazabilidad paso a paso.
- Exportación de auditoría.

## Duración
4 a 6 semanas.

## Entregables
- Ambiente piloto operativo.
- Dashboard de runs + traza.
- Baseline de seguridad aplicada.
- Informe de resultados y plan de expansión.

## KPIs de éxito
- 90%+ runs con trazabilidad completa.
- Reducción de tiempo de diagnóstico de incidentes.
- Sponsor técnico confirma valor operativo.

## Criterios de no-go
- Cliente sin workflow real.
- No existe owner de negocio/técnico.
- Requiere customización fuera de alcance MVP.
