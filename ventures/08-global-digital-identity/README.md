# 08 — Global Digital Identity

## Visión
Construir una identidad digital verificable, portable y programable para personas y organizaciones.

## Problema
La identidad está fragmentada entre gobiernos, plataformas y proveedores privados, con baja interoperabilidad y escaso control del usuario.

## Propuesta
Una identidad digital con credenciales verificables, reputación, permisos y portabilidad entre servicios.

## Cliente inicial
- Fintech
- Healthtech
- Educación
- Plataformas de talento y contratación

## Modelo de negocio
- API por verificación
- Wallet premium
- Infraestructura B2B2C
